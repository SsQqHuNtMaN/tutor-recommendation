## 🌟 它能为你做什么

你不需要自己整理教师名单、逐页查论文，也不用学习项目里的命令。只要准备个人材料并告诉 Agent 想申请哪里，它会完成剩余工作。

- 🎯 **匹配导师**：结合你的研究经历、兴趣和申请偏好筛选教师，并把命中方向按数学方法、统计方法、AI 方法等画像分组解释。
- 👤 **切换学生画像**：为不同申请方向保留独立画像，在看板标题栏随时切换，结果和联系记录不会混在一起。
- 🔎 **核验信息**：查看学院官网、教师主页和适合该学科的论文来源；计算机方向使用 DBLP，数学/统计方向使用官网论文、zbMATH 等证据。
- 🏫 **补充新院校**：目标学校尚未收录时，Agent 会自行接入官方教师目录后再运行匹配。
- 🧑‍🏫 **集中看详情**：在一个页面查看教师方向、匹配理由、论文证据和主页入口。
- 🗓️ **跟进套磁日历**：按四周日历记录已发、官网回复、添加微信、约面试等进展。
- 📊 **掌握申请节奏**：通过学校与学院概览查看教师数、已套磁数、有进展数和面试数。

你的简历、画像、匹配结果和联系记录默认保存在本机。Agent 不会替你发送邮件或修改套磁状态，最终选择始终由你决定。

## 🔬 推荐为什么可核验

- **方向先有依据**：优先结论必须能回到学院名录、教师主页或官方附件中的明确研究方向。
- **论文按学科补充**：计算机目标使用 DBLP；数学与统计目标优先整理官网论文和 zbMATH Open，并可在本机配置后使用 OpenAlex 补全跨学科记录。
- **同名不会直接认领**：作者姓名只用于召回候选；机构、ORCID、官网题名或已确认作者 ID 达到身份门槛后，外部论文才会参与计分。
- **证据强弱分开显示**：低置信 arXiv、身份不确定数据库结果和自动 WebSearch 只作为线索，不能单独制造高推荐；工作簿和看板保留方向分组、分来源得分与警告。

## 🚀 如何使用 Coding Agent

### 1️⃣ 准备个人材料

把已有材料放入：

```text
user_private/source/
```

简历、成绩单、申请表、项目说明或个人陈述都可以。材料不完整也没关系，Agent 会先整理能够确认的信息，再集中询问缺少的内容。

### 2️⃣ 告诉 Agent 你的目标

在项目根目录打开 Codex 或其他 Coding Agent，然后发送：

```text
请读取 user_private/source/ 中的材料，为我匹配【学校】【学院】的导师。

先整理我的学生画像，并把需要确认的研究方向、关键词和申请偏好一次列给我。
我确认后，请自行收集学院官网与教师主页信息，补充论文证据，完成导师匹配和质量检查。
如果项目还不支持这所学校，请自行添加，不需要让我操作代码。
完成后请打开套磁面板。
```

你也可以补充这些偏好，让推荐更贴近真实申请：

- 想重点申请的研究方向
- 明确不考虑的方向
- 学校、地区或学院范围
- 更看重学术契合、招生可能性，还是联系进展

### 3️⃣ 确认学生画像

Agent 会先把材料整理成画像草稿，再请你确认。重点检查：

- ✅ 研究兴趣是否准确
- ✅ 项目、论文和技能是否遗漏
- ✅ 关键词的重要程度是否符合你的想法
- ✅ 数学、统计、AI 等方向分组是否覆盖你愿意探索的交叉区域
- ✅ 是否存在不想申请的方向

确认后回复“画像无误，继续匹配”即可。

如果你同时准备不同方向，可以让 Agent 分别创建画像，例如“计算机 × AI”和“数学 × AI”。看板标题栏的“学生画像”选项可以手动切换；每个画像只展示自己的教师结果、日历和联系状态。

### 4️⃣ 查看导师推荐

Agent 完成后会打开本地套磁面板。你可以在里面：

- 按学校、学院、推荐等级和联系状态筛选教师
- 查看“强烈建议”“可以考虑”“暂不优先”及对应理由
- 对照官网方向、论文证据和教师主页进行复核
- 在右侧详情栏记录套磁状态、回复内容和面试时间

### 5️⃣ 用日历持续跟进

四周日历会把联系进展放在同一个时间视图中：

- 📤 已发套磁
- 💬 官网回复
- 🟢 添加微信
- 🎤 约面试与具体时间
- 📝 考核、名额已满等后续状态

点击日期即可查看当天教师；学校与学院进度概览不受四周范围限制，适合快速判断下一步应该联系谁。

### 🖱️ 以后再次打开

Windows 用户双击根目录的 [`start_viewer.bat`](start_viewer.bat) 即可。已经启动时会直接打开现有页面；端口被占用时会自动换到可用地址，不会清空联系记录。

需要了解更多时，可以继续直接问 Agent，例如：

```text
帮我查看还没有联系的高匹配导师。
帮我检查本周有哪些回复和面试。
帮我为新的学校添加导师匹配。
帮我重新核验这位老师的研究方向和近期论文。
```
